Mobile Test Automation Engineer for AI Training

Location: Remote Type: Full-time or Part-time Contract

In the era of Generative AI, code is being written faster than ever before—but speed means nothing without stability. At Booz Allen, we are seeking a meticulous Mobile Test Automation Engineer to ensure that our AI-generated mobile applications are robust, bug-free, and user-ready. You will be responsible for "automating the automation," defining how AI understands quality assurance.

You won't just be finding bugs; you will be teaching our AI models how to hunt for them. By creating comprehensive test suites using Appium, Espresso, and XCUITest, and by evaluating AI-generated test scripts, you will help build a future where AI can self-correct and verify its own code against rigorous quality standards.

Benefits

  • This is a full-time or part-time remote position with a flexible schedule.
  • Master the latest mobile testing frameworks and cloud device farms (e.g., Sauce Labs, BrowserStack).
  • Work at the cutting edge of "AI in QA," shaping how machines approach software testing.
  • This is a contract position with a highly competitive rate, starting at $80+ USD per hour.

Responsibilities

  • Develop scalable and maintainable automation frameworks for iOS and Android using Appium, Espresso, and XCUITest.
  • Create diverse, complex test scenarios (edge cases, regression, UI/UX) to train AI models on proper testing logic.
  • Review AI-generated test code to identify logic gaps, "flaky" tests, or inefficient selectors.
  • Integrate automated tests into CI/CD pipelines (Jenkins, GitHub Actions, Bitrise).
  • Write "Chain of Thought" reasoning to explain why a specific test failed and how to debug it, providing data for the model to learn from.
  • Perform automated performance profiling (memory leaks, battery drain) on mobile devices.

Requirements

  • 3+ years of experience in Mobile QA Automation.
  • Strong proficiency in Appium (with Java, Python, or JS) and native tools (Espresso/Kaspresso for Android, XCUITest for iOS).
  • Solid coding skills in at least one major language (Java, Kotlin, Swift, or Python).
  • Experience with cloud-based device farms (Sauce Labs, BrowserStack, or AWS Device Farm).
  • Understanding of CI/CD concepts and version control (Git).
  • A "breaker" mindset—you enjoy finding edge cases that crash applications.
  • Experience with Behavior Driven Development (BDD) tools like Cucumber is a plus.
