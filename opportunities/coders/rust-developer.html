Rust Developer for AI Training

Location: Remote Type: Full-time or Part-time Contract

AI models provide the intelligence, but Rust provides the unbreakable, high-performance foundation. At Booz Allen, we are looking for a skilled Rust Developer to build the blazingly fast, memory-safe, and concurrent infrastructure that powers our AI models at scale.

While our models are prototyped in Python, they are delivered to the world through mission-critical Rust microservices. Your role is to engineer these back-end systems—the APIs, data processing engines, and orchestration services that demand maximum performance and reliability. If you are passionate about systems programming, memory safety, and fearless concurrency, this is your chance to build the core of a next-generation AI platform.

Benefits

  • This is a full-time or part-time remote position with a flexible schedule.
  • Build the high-performance, fault-tolerant infrastructure that serves our most advanced AI models.
  • Leverage Rust's ownership model and fearless concurrency to build truly reliable and parallel systems.
  • This is a contract position with a highly competitive rate, starting at $80+ USD per hour.

Responsibilities

  • Develop high-performance, memory-safe microservices and APIs in Rust for model serving, ensuring minimal latency and maximum uptime.
  • Build concurrent data processing pipelines that efficiently handle high-volume data streams for our AI systems.
  • Create efficient and safe bindings to Python-based ML libraries where high-performance Rust components are required.
  • Optimize code for CPU and memory efficiency, leveraging Rust’s zero-cost abstractions in a cloud environment.
  • Write robust infrastructure tooling and automation scripts to support our MLOps and SRE teams.
  • Collaborate with Machine Learning Engineers to design clean, fast, and type-safe API contracts for model integration.

Requirements

  • 2+ years of professional software development experience with a strong focus on the Rust programming language.
  • A deep understanding of Rust's ownership model, borrowing, lifetimes, and concurrency primitives (e.g., Arc, Mutex, async/await).
  • Proven experience building and deploying high-performance RESTful or gRPC APIs in Rust (using frameworks like Actix-web, Axum, or Tonic).
  • Familiarity with the standard Rust toolchain (e.g., cargo, rustc, clippy).
  • Knowledge of containerization (Docker) and experience deploying applications to a cloud environment (AWS, GCP, Azure).
  • A strong foundation in systems programming principles and writing clean, idiomatic, and well-tested Rust code.
  • Familiarity with Kubernetes and experience with Python interoperability (e.g., PyO3) are strong pluses.
