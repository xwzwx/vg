Android Developer for AI Training

Location: Remote Type: Full-time or Part-time Contract

The future of AI isn't just in the cloud—it's in the palm of your hand. At Booz Allen, we are seeking a talented Android Developer to help bridge the gap between large-scale AI models and mobile deployment. You will be responsible for creating the native interfaces and optimization layers that allow our advanced AI to function seamlessly on mobile devices.

In this role, you will work on the bleeding edge of "Edge AI," ensuring that our models are not only accurate but also performant, battery-efficient, and responsive. You will write high-quality Kotlin code to help train our AI coding assistants and build the applications that deploy these models to users in the field.

Benefits

  • This is a full-time or part-time remote position with a flexible schedule.
  • Work with the latest mobile technologies, including Jetpack Compose and on-device ML accelerators.
  • Contribute to the training data that powers the next generation of AI coding tools.
  • This is a contract position with a highly competitive rate, starting at $100+ USD per hour.

Responsibilities

  • Design and build advanced native Android applications using Kotlin and the latest Android SDKs.
  • Collaborate with AI/ML researchers to implement model inference on Android devices (using TensorFlow Lite or similar).
  • Create robust, clean, and testable code samples to train and evaluate AI coding models.
  • Optimize application performance, focusing on memory management, battery usage, and latency for AI features.
  • Implement modern Android architecture patterns (MVVM/MVI) and dependency injection (Hilt/Dagger).
  • Review AI-generated Android code to ensure accuracy, security, and best practices.

Requirements

  • 3+ years of professional Android development experience.
  • Expert proficiency in Kotlin and a strong understanding of Java legacy code.
  • Deep knowledge of the Android lifecycle, concurrency (Coroutines/Flow), and networking.
  • Experience with modern UI toolkits, specifically Jetpack Compose.
  • Familiarity with mobile ML frameworks (ML Kit, TFLite) is a significant plus.
  • Strong understanding of software design patterns and clean architecture.
  • A passion for mentoring and teaching—helping the AI "learn" how to write better Android code.
