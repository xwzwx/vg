React Native Developer for AI Training

Location: Remote Type: Full-time or Part-time Contract

React Native allows developers to "learn once, write anywhere," and now it is time for AI to learn it too. At Booz Allen, we are seeking a React Native Developer to help our AI models master the art of bridging JavaScript with native mobile architecture. You will be responsible for generating the data that teaches AI how to build performant, native-feeling apps using the React ecosystem.

In this role, you will focus on modern TypeScript, Functional Components, and the nuances of the React Native Bridge (and the New Architecture). You will create code samples and evaluate model outputs to ensure the AI understands not just React logic, but mobile specific constraints like native linking, safe area handling, and platform-specific styling.

Benefits

  • This is a full-time or part-time remote position with a flexible schedule.
  • Work with the latest React Native technologies, including Expo, Reanimated, and the New Architecture (Fabric/TurboModules).
  • Shape the future of AI-assisted development by defining code quality standards for the React community.
  • This is a contract position with a highly competitive rate, starting at $120+ USD per hour.

Responsibilities

  • Develop comprehensive React Native components and applications using TypeScript to create high-quality training datasets.
  • Demonstrate best practices for state management (Redux Toolkit, Context API, Zustand) and side-effect handling.
  • Write clear examples of "Native Modules" to teach the AI how to interface JavaScript with Swift and Kotlin code.
  • Review AI-generated code for performance bottlenecks, specifically regarding re-renders and bridge traffic.
  • Create "Chain of Thought" documentation to explain the lifecycle of a React Native app compared to a standard Web React app.
  • Implement smooth animations using React Native Reanimated to show the AI how to achieve 60fps UI.

Requirements

  • 3+ years of professional experience with React Native and TypeScript.
  • Strong command of React Hooks (useEffect, useCallback, useMemo) and Functional Components.
  • Experience with navigation libraries, specifically React Navigation.
  • Familiarity with both the Expo ecosystem and the bare React Native CLI.
  • Understanding of native build tools (Xcode and Android Studio) and how to debug native crashes.
  • Knowledge of testing libraries like Jest and React Native Testing Library.
  • Ability to explain complex concepts, such as the Shadow Thread vs. the JS Thread, in simple terms.
