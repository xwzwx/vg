Python Expert for AI Training

Location: Remote Type: Full-time or Part-time Contract

Python is the undisputed language of artificial intelligence, but leveraging it to its full potential requires true mastery. At Velocity Global, we are seeking a Python virtuoso—a developer who thinks in Pythonic idioms and is passionate about performance—to elevate our entire AI training infrastructure.

In this role, you will be the go-to expert for the most complex Python challenges. You will write the critical, high-performance code that processes massive datasets and underpins our machine learning pipelines. Your work will be the foundation that enables our AI models to learn faster, more efficiently, and at a greater scale.

Benefits

  • This is a full-time or part-time remote position with a flexible schedule.
  • Apply your deep Python expertise to solve fascinating, large-scale problems in the AI/ML domain.
  • Become the technical authority and thought leader for Python best practices within a team of talented engineers.
  • This is a contract position with a highly competitive rate, starting at $70+ USD per hour.

Responsibilities

  • Write high-performance, idiomatic Python code for data-intensive applications and machine learning pipelines.
  • Develop and maintain core libraries, utilities, and frameworks that are used across the AI engineering team.
  • Profile and optimize critical code paths to eliminate performance bottlenecks in data processing and model training.
  • Architect and implement complex data processing scripts for cleaning, transforming, and preparing large datasets.
  • Mentor other engineers on advanced Python features, code quality, and performance optimization techniques.
  • Set and maintain the highest standards for Python code quality, testing, and documentation.

Requirements

  • Extensive professional experience and demonstrated mastery of the Python programming language.
  • Deep understanding of Python's internals, including memory management, concurrency (asyncio, multiprocessing), and performance characteristics.
  • Expert-level knowledge of the scientific Python stack (e.g., NumPy, Pandas, SciPy) and its performance considerations.
  • Proven ability to write clean, maintainable, and highly efficient code for complex problems.
  • Experience working in a collaborative environment using version control (Git) and code reviews.
  • A passion for elegant solutions, code craftsmanship, and sharing knowledge with others.
  • Experience supporting machine learning workflows is a significant advantage.
