C# Developer for AI Training

Location: Remote Type: Full-time or Part-time Contract

AI is reshaping the technology landscape, but it requires a solid, scalable foundation to operate effectively in the enterprise. At Booz Allen, we are seeking a talented C# Developer to bridge the gap between experimental machine learning models and the rigorous world of the Microsoft ecosystem.

Your role will be to engineer the critical backend services and integration layers that bring AI to life. You will leverage the modern .NET ecosystem to wrap complex AI logic into performant, secure, and consumable APIs. If you are passionate about clean architecture, high-performance computing, and integrating next-gen intelligence into established workflows, this is the challenge you’ve been looking for.

Benefits

  • This is a full-time or part-time remote position with a flexible schedule to fit your lifestyle.
  • Work at the intersection of AI innovation and enterprise-grade software engineering.
  • Collaborate on high-impact projects that deploy AI capabilities into secure, real-world environments.
  • This is a contract position with a highly competitive rate, starting at $120+ USD per hour.

Responsibilities

  • Design and develop high-performance RESTful and gRPC APIs using ASP.NET Core to serve AI model predictions.
  • Integrate Machine Learning models (ONNX, ML.NET, or Python-wrapped services) into robust .NET production environments.
  • Build scalable microservices that interact with cloud infrastructure (Azure) and message brokers (e.g., Azure Service Bus, RabbitMQ).
  • Optimize application performance and memory management to handle high-throughput inference requests.
  • Collaborate with Data Scientists to containerize and deploy models using Docker and Kubernetes.
  • Write clean, testable, and maintainable C# code that adheres to industry best practices and security standards.

Requirements

  • 3+ years of professional experience in software development with C# and the .NET Framework / .NET Core.
  • Strong proficiency in building web APIs with ASP.NET Core.
  • Experience with cloud platforms (preferably Azure) and understanding of cloud-native design patterns.
  • Solid understanding of asynchronous programming, dependency injection, and object-oriented design principles.
  • Familiarity with ML concepts or libraries (ML.NET, ONNX Runtime) is a significant plus.
  • Experience with containerization (Docker) and CI/CD pipelines (Azure DevOps or GitHub Actions).
  • A commitment to writing high-quality, self-documenting code and comprehensive unit tests.
